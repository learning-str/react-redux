# memo
- stateオブジェクトがすべての変数を持っている。
- actionsオブジェクトが変更部分のオブジェクトを持っている。
- pure function(純粋関数)は引数を変更しない。impure function(副作用?)は引数を変更する。reducer function はpure function である必要がある。
